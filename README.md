# AckAsian mod

[<img alt="modrinth" height="56" src="https://cdn.jsdelivr.net/npm/@intergrav/devins-badges@3/assets/cozy-minimal/available/modrinth_vector.svg">](https://modrinth.com/project/ackasian)

Simple mod that adds the [AckAsian](https://www.ackasian-food.com/) menu to Minecraft

It adds:
- **The rice crop:** New plant that can be used for multiple recipes !
- **Flour and Dough:** Basic elements to make other dishes
- **Noodles and rice noodles:** New alternative for a fast, cheap and farmable meal! Can be used for other recipes
- **3 new dishes:** The Banh Mi, the Samosa and the Spring Rolls.
- **4 new special dishes:** The Bo Bun, the Coconut Chicken Curry, the Fried Rice and the Pad Thai. Each one gives different potion effects !

This mod is available in English and French
